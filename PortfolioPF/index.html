FizzBuzz Coding Challenge

FizzBuz is a coding challenge invented by Imran Ghoryin in 2007. Multiples of the first entered number print Fizz. Multiples of the second entered number print Buzz. Multiples of both entered numbers print FizzBuzz.

  • Built with HTML, CSS, Bootstrap, JQuery and JavaScript

                                                //Step 1: Get the data
                                                var fizzNum = Number($("#fizzNum").val());
                                                var buzzNum = Number($("#buzzNum").val());

                                                var arrOutput = new Array();

                                                //Step 2: Do something with the data
                                                for (var loop = 1; loop <= 100; loop++) {
                                                    if ((loop % fizzNum == 0) && (loop % buzzNum == 0)) {
                                                        //FIZBUZZ condition
                                                        arrOutput.push("FIZZ-BUZZ");
                                                    }
                                                    else if (loop % fizzNum == 0) {
                                                        //FIZZ condition
                                                        arrOutput.push("FIZZ");
                                                    }
                                                    else if (loop % buzzNum == 0) {
                                                        //BUZZ condition
                                                        arrOutput.push("BUZZ");
                                                    }
                                                    else {
                                                        //LOOP condition
                                                        arrOutput.push(loop);
                                                    }
                                                }

Taco Cat Coding Challenge

  • The Taco Cat coding challenge reverses a string of letters and checks if the word is a palindrome. An example of a palindrome is "tacocat".

  • Built with HTML, CSS, Bootstrap, JQuery and JavaScript

Enter a string to be reversed:
Reversed word result:


                                                //1. Get user input
                                                var word = $("#inputReverse").val();
                                                wordLowCase = word.toLowerCase();

                                                //Step 2.Validate the data
                                                if (word.length < 1) {
                                                    alert("Your entry is empty, please enter a word.");
                                                    return;
                                                }

                                                //2.Split word into letters, count letters and reverse the string
                                                wordLength = word.length;

                                                for (i = wordLength - 1; i >= 0; i--) {
                                                    wordReverse += wordLowCase[i];
                                                }

                                                if (wordLowCase == wordReverse) {
                                                    $("#tacoCatOutput").text(word + " is a palindrome.");
                                                }
                                                else {
                                                    $("#tacoCatOutput").text(word + " is not a palindrome.");
                                                }

                                                //3. Output results
                                                $("#outputReverse").val(wordReverse);

Sunset Hills Coding Challenge

  • The height is entered for a grouping of six buildings. As the sun will set on the left side of the screen, some buildings can see the sunset while others are blocked by taller buildings. Based on the heights entered, the program determines which buildings can see the sunset.
  • This was built with HTML, CSS, Bootstrap, JQuery and JavaScript

                                                //1. Get user input into an array.
                                                var buildings = [];
                                                var seeSunset = [];

                                                buildings[0] = Number($("#input0").val());
                                                buildings[1] = Number($("#input1").val());
                                                buildings[2] = Number($("#input2").val());
                                                buildings[3] = Number($("#input3").val());
                                                buildings[4] = Number($("#input4").val());
                                                buildings[5] = Number($("#input5").val());
                                                //var buildings[0] = String($("#input1").val());
                                                //var buildings[1] = Number($("#input2").val());


                                                //2.Check which buildings can see the sunset.

                                                //Run through all 6 buildings
                                                for (b = 5; b >= 0; b--) {
                                                    //check if building can see the sunset
                                                    for (i = b - 1; i >= 0; i--) {
                                                        if (buildings[b] > buildings[i]) {
                                                            seeSunset[b] = buildings[b];
                                                            //seeSunset[b] = "Yes";
                                                        }
                                                        else {
                                                            seeSunset[b] = "";
                                                            noSunsetCount += 1;
                                                            break;
                                                        }
                                                    }
                                                }

                                                //Building 0, First building can always see the sunset
                                                seeSunset[0] = buildings[0];

                                                //Number of buildings that can see the sunset
                                                numBuildings = buildings.length - noSunsetCount;

                                                //3. Output results
                                                //$("#output").val(buildings[5]);
                                                //$("#output").val(seeSunset + " " + numBuildings);
                                                $("#sunsetHillsOutput").text(numBuildings + " buildings can see the sunset: " + (seeSunset.join(" ")));  //.join to get rid of the commas
